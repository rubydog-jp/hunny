# 中断方法

特に何も気にせず中断することができます。

# 再開方法

「手順4.ブランチ」以降は gitコマンドを打つ時に 条件があります。  
中断したときにターミナルを閉じたり、パソコンの電源を消していたときは、
以下を入力しなおしてください。

```
$ cd hunny
```

<br />

- Macの場合 `xxx/hunny $`
- Windows `¥xxx/hunny>`

のように `/hunny` と表示されていれば OK です。
