# 中断方法

特に何も気にせず中断することができます。

<br />

# 再開方法

### 手順1~3

特に何も気にせず再開することができます。

### 手順4~7

ターミナルやパワーシェルのアプリを閉じたときは  
再開するときに 以下を入力しなおしてください。

```
$ cd hunny
```

<br />

- Macの場合 `xxx/hunny $`
- Windows `¥xxx/hunny>`

のように `/hunny` と表示されていれば OK です。
